# 🏁 Jogo de Xadrez Completo com Python

Um jogo de xadrez completo desenvolvido em Python com interface gráfica, inteligência artificial Stockfish e todas as regras oficiais do xadrez.

![Gameplay](Direção%20da%20peça.png)

## 🌟 Recursos

- ✅ Interface gráfica com Pygame
- 🤖 Inteligência Artificial Stockfish (níveis 0-20)
- ♟️ Todas as regras do xadrez: Roque, en passant, promoção
- 🖱️ Controles intuitivos: clique para mover peças
- 📊 Indicadores visuais: xeque, movimentos válidos
- 💾 Salvar e carregar partidas (formato PGN)
